Custom GPTs *should be* the best thing of ChatGPT.

1. You give it a “pre-prompt” as instructions it must follow before the chat.2. You give it knowledge it must know before the chat.

For example:

1. Custom instructions on how to write the best contracts.2. A long PDF of “best written contract” from a trusted source.3. Now as soon as you start a chat → perfect contracts, instantly.

You can duplicate your taste & experience in one “GPT” **anyone can use** (or you keep it private). You can even share them online, and get as much as 800,000+ users!

But it had one very big problem: **the model**.

You know that ChatGPT has many models, that I can summarize quickly as:

- o3: best to reason complex tasks.- gpt-4.1: best for quick tasks.- gpt-4.5: best to write.

You almost always want to pick o3 over any other model.

But custom GPTs used to not only support **gpt-4o** (the worst model).

And this is over. **You can now select any model**.

Here’s my step-by-step guide on:

1. Creating a custom GPT the right way.2. Selecting the right model, the right way.3. Finally going beyond simple copy & pasting prompts.

## #1 - How to create your GPT.

![Top left menu opener → Click on “GPTs”.](https://substack-post-media.s3.amazonaws.com/public/images/7de7703c-ef05-4f92-862e-9e93d0433eea_1200x800.jpeg)
*Top left menu opener → Click on “GPTs”.*

![Click on “+ Create”.](https://substack-post-media.s3.amazonaws.com/public/images/396df770-aa1c-4327-b385-632d1ddcae2c_1200x800.jpeg)
*Click on “+ Create”.*

You end up on this menu to **customize** your **GPT**:

![](https://substack-post-media.s3.amazonaws.com/public/images/8f065323-078c-4eea-9a89-cbf0be97b1fe_1684x1984.png)

1. Name: just pick any name. Useful when share publicly / when you create a lot of them.2. Description: a one-liner explaining what’s the GPT doing.Again, very useful when shared to a team / navigating.3. Instructions: the prompt before the chat. Most important part of the GPT. I will write a whole section on it.4. Conversation starters: to give an example of a first message.I usually do “Click here” or leave it blank.5. Knowledge: where you upload documents that the GPT can read and use to answer questions based on their content. It’s context or examples.6. Recommended model*: newest addition. Choose the model.o3 for reasoning / gpt-4.1 for quick tasks / gpt-4.5 to write7. Capabilities: which ChatGPT feature you want to turn on.Just select everything.8. Actions: to connect external tools.So it can do things—like search the web, send emails, fetch data, or run code; beyond just chatting. Too technical for now for people like us.

Let’s build one together.

## #2 - Hook Generator GPT

So far, I gave it a name “Hook Generator GPT” and a description “to write better hooks”. For **3)** **Instructions**, now comes the real work.

I usually start like this:

- I open my Prompt Maker.- I explain very simply the prompt I need.- I test it, change the prompt if needed, until I enjoy it.

![Make sure to use “o3” inside Prompt Maker.](https://substack-post-media.s3.amazonaws.com/public/images/4d0d72e0-15ff-4011-b0c4-90f70f0b7f03_1200x800.jpeg)
*Make sure to use “o3” inside Prompt Maker.*

I shared this prompt to simply describe what I need, with an example:

> I want a prompt to write hooks for my social media post.1 - only sentences below 55 characters2 - hooks are 2 lines (with a line break in between)3 - the first line must make you feel something negative (like a gap)4 - the second line must be an invitation to read moreFor example:///You don't know how to do SEO anymore.But this one thing ruins your website:///

And my GPT “**Prompt Maker**” gave me a solid first prompt (our chat is [here](https://chatgpt.com/share/685ecf57-6f1c-800f-b425-8537424e7538)).

I kept iterating like this:

![](https://substack-post-media.s3.amazonaws.com/public/images/fb6d1727-6aba-4f79-9d1d-095ef13a3342_2082x1790.png)

Now I can copy it, and paste it on our GPT panel from before:

![](https://substack-post-media.s3.amazonaws.com/public/images/14f82d0f-1828-4ff9-b15b-ed9acce78cd6_1200x800.jpeg)

For conversation starters, I often time do this:

![](https://substack-post-media.s3.amazonaws.com/public/images/c2901d2a-e99e-4ea7-bb1c-1c6fff1d4ef7_1678x296.png)

The reason why is that I tend to forget how to use each and every GPT. But when you just “Start”, the GPT will explain to me what to do.

The “Knowledge” part is quite important: here, I would simply compile my favorite hooks on a txt. file from a google doc. Like this for example: https://docs.google.com/document/d/1PiJ7IFSy0T466SokEr-QWb1j4s1_nNyZdbDzV9T7TOM/edit?usp=sharing.

![](https://substack-post-media.s3.amazonaws.com/public/images/57ac6427-ba7d-4f5d-9da0-79401d250577_1242x346.png)

Now you can simply:

1. Create it (top right). Select on the “GPT Store”.2. Test it using the GPT link, like: https://chatgpt.com/g/g-685d6f8f1c048191907f3e0cfd2b0dbf-hook-generator-gpt.3. Update it until it works! And so far, it’s already working for me.
