## How to install Claude in Excel.

First, you must have Microsoft Excel installed.

Quick tip: I downloaded Excel for free *(Mac)* using this official Microsoft link:
→ https://go.microsoft.com/fwlink/p/?linkid=525135.

Once you have it installed with a free Microsoft account, install Claude:

1. Go to the Microsoft Marketplace listing.2. Click “Get it now” → Install the Claude add-in.3. Open Excel → Tools → Add-ins (Mac) or Home → Add-ins (Windows).4. Sign in with your Claude account. If you still don’t have one, go get one.5. PS: You do need to pay for Claude ($17/mo). It’s worth it.
