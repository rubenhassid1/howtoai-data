## 4. A vibe coding model: bye-bye Lovable?

This newsletter is about “**How to AI**” with English, not code.

So I don’t like talking about coding anything.

But here, I have to say ChatGPT surprised me.

Just like the now-famous Lovable, you can ***vibe code**** a tool quite easily.

**vibe coding: writing with English what you want to code, without coding.*

![](https://substack-post-media.s3.amazonaws.com/public/images/45d4093d-bc0c-4b2e-8657-6d91abb8cf06_3456x1984.png)

## 5. Best of GPT-5 prompts (to copy & paste).

**Prompt template**

```
`Act like a {ROLE best suited to this task}. 

Goal: {OUTCOME in one sentence}

Inputs: GOAL | CONTEXT (audience, use-case)

PREFERENCES (tone, length, language)

CONSTRAINTS (time/budget/policy)

DATA/LINKS | DEADLINE & LOCALE
If anything crucial is missing, ask ≤3 targeted questions; else proceed with explicit assumptions.

Process: (1) Clarify & assumptions → (2) Brief plan → (3) Step-by-step solution → (4) 2–3 alternatives w/ pros/cons → (5) Recommendation & next steps.

Output format:
- TL;DR (3–5 bullets)
- Detailed steps/results (concise)
- Artifacts if useful (code/tables/checklists)
- Sources (titles + dates) for time-sensitive claims
- Open questions + confidence

Style: clear, practical, localized, no fluff; note dates/timezone when relevant; follow safety policies.

Take a deep breath and work on this problem step-by-step.`
```
